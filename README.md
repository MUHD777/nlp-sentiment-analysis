## Project Overview

This project demonstrates **sentiment analysis on social media data** with a focus on **time series trends**. The goal is to understand how sentiment changes over time, while performing data cleaning to prepare the dataset for analysis.

- Analyze text data for **positive, negative, and neutral sentiment**  
- Explore **temporal trends** in social media activity  
- Generate visual insights for **decision-making**  

---

## Tools & Libraries

- **Python**  
- **Pandas** (data handling)  
- **NLTK** (text preprocessing & tokenization)  
- **TextBlob** (sentiment analysis)  
- **Matplotlib** & **Seaborn** (visualization)  

---

## Workflow (Time Series Section)

1. **Load Data**  
   - Import CSV containing social media posts with timestamps  

2. **Minimal Data Cleaning**  
   - Remove duplicates  
   - Handle missing values (if any)  
   - Keep original columns for reference  

3. **Preprocess Text**  
   - Tokenization using NLTK  
   - Remove stopwords  
   - Lemmatization for meaningful base words  

4. **Sentiment Analysis**  
   - Compute polarity using TextBlob  
   - Classify each post as **Positive, Negative, Neutral**  

5. **Time Series Analysis**  
   - Convert timestamps to datetime objects  
   - Extract **year, month, day, hour** features  
   - Aggregate sentiment over time  
   - Visualize trends using line charts
